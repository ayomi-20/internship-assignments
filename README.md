# Python Mini Projects

This repository contains four simple Python programs files:

- Simple Calculator
- Student Grading System
- VAT Calculator
- Inventory Management System

# Simple Calculator

**File name** `calculator.py`

This program performs basic arithmetic operations. The user enters two numbers and chooses an operation to perform.

## Features

- Addition
- Subtraction
- Multiplication
- Division

# Student Grading System

**File name** `grading_system.py`

This program accepts marks for Mathematics, English, Science, and Social Studies. It calculates the average mark and displays the student's grade.

## Subjects

- Mathematics
- English
- Science
- Social Studies

# VAT Calculator

**File name** `vat_calculator.py`

This program accepts a sales amount, calculates VAT at 18%, and displays the total amount to be paid.

## Formula

VAT = 18% × Sales Amount

Total Amount = Sales Amount + VAT

# Inventory Management System

**File name** `inventory_management_system.py`

This program uses a list and a dictionary to store products and their quantities.

## Features

- Stores product names in a list.
- Stores product quantities in a dictionary.
- Displays the inventory.

# Technologies Used

- Python 3
